2. Welche Daten wir verarbeiten

Diese Website erhebt keine personenbezogenen Daten auf unseren Servern. Es gibt keine Nutzerkonten, kein Login und kein serverseitiges Tracking.

Die folgenden Funktionen speichern Daten ausschließlich lokal in deinem Browser (localStorage):

  • Meine Teilnahmen: Welche Gewinnspiele du als "teilgenommen" markiert hast (Titel, Kategorie, Deadline, Veranstalter). Diese Daten verlassen deinen Browser nicht.
  • Mein Profil: Deine freiwillig eingetragenen Daten (Vorname, Nachname, E-Mail, Wohnort, PLZ, Geburtsdatum) für die AutoFill-Funktion. Diese Daten verlassen deinen Browser nicht.
  • Benachrichtigungen: Ein Log bereits gesendeter Deadline-Erinnerungen zur Vermeidung von Duplikaten.

Du kannst diese Daten jederzeit löschen, indem du den Browser-Cache / localStorage löschst (Browser-Einstellungen → Daten löschen).
